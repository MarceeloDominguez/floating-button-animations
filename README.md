## Run Locally

Clone the project

```bash
  https://github.com/MarceeloDominguez/floating-button-animations.git
```

Go to the project directory

```bash
  cd floating-button-animations
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
